## Getting Started

### Pre-requisits:

-   You need a mongodb server running localy on port 27017

First, install the dependencies:

```bash
npm install
```

Then, run the application:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser and have fun with Hydraca.
